# IPL Auction Strategy Analysis & Performance Dashboard

This project analyzes historical IPL data to derive insights for building a strong and balanced team during the IPL auction.

## Project Overview
The analysis evaluates batting performance across different phases of the game and identifies impactful players based on statistical metrics.

## Tools Used
Python
Pandas
NumPy
Matplotlib 
Seaborn
Power BI

## Key Features
- Data cleaning and preprocessing of IPL datasets
- Feature engineering for strike rate, boundary percentage, and scoring patterns
- Phase-wise analysis across Powerplay, Middle Overs, and Death Overs
- Interactive Power BI dashboard for player comparison and performance insights

## Outcome
The dashboard enables data-driven decision making for IPL auction strategy and player selection.
